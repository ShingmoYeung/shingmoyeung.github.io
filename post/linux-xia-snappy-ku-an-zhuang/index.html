Linux 下 Snappy库安装

2018-11-24 13:30

下载

1. 本文使用 Git 安装必须机器上配置好了Git。Git配置详见http://blog.csdn.net/sdoyuxuan/article/details/78751494
 2. https://github.com/google/snappy 点击Download 复制粘贴SSH

 3. git clone https://github.com/google/snappy.git

安装

预装 cmake 可以搜下cmake 源码安装

tar -xvzf google-snappy-1.1.7-3-ge69d9f8.tar.gz 
mkdir build cd build && cmake ../ && make 
mv snappy-stubs-public.h /usr/include

使用

g++ test.cc -lsnappy

即可
